Installation

Skills are automatically detected by Claude Code after copying.

All skills

# Clone the repo
git clone https://github.com/cuellarfr/design-skills.git

# Copy all skills to Claude Code
cp -r design-skills/*/ ~/.claude/skills/

Single skill

# Example: install only design critique
cp -r design-skills/design-critique \
  ~/.claude/skills/

Skill Architecture

Every skill follows a consistent, progressive-depth structure.

skill-name/
├── SKILL.md              # Main file — works standalone
│
├── references/           # Deep reference material
│   ├── topic-a.md
│   └── topic-b.md
│
├── templates/            # Fillable deliverable templates
│   └── report-template.md
│
└── examples/             # End-to-end walkthroughs
    └── walkthrough.md
  • Opinionated defaults Specific numbers, benchmarks, and rubrics — not "consider…" language
  • Actionable over theoretical Every principle includes a concrete recommendation
  • Before/after examples Show what bad and good look like, with scoring
  • Progressive depth Main file is comprehensive standalone; references go deeper
  • Grounded in sources Built from established books and frameworks, not invented heuristics
